A basic mensuration calculator made in JavaScript.

## Features

- Perimetre - Reactangle, Square, Circle, Triangle, Parallelogram
- Area - Reactangle, Square, Circle, Triangle, Parallelogram
- Volume - Cuboid, Cube, Sphere, Cone, Cylinder
